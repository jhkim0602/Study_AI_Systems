# StudyNote Hub

이 폴더가 실제 학습 자료의 기준 허브입니다.

정리 원칙
- 주차별 학습 노트는 각 `weekXX/` 폴더 안에서 관리합니다.
- 실습 노트북도 같은 주차 폴더 안에 함께 둡니다.
- 데이터와 이미지 자료는 해당 주차 폴더의 `data/`, `assets/`에 둡니다.
- 링크는 가능한 한 같은 주차 폴더 안에서 닫히도록 유지합니다.
